# Mineradio

Mineradio 是一款以电影镜头、粒子视觉和歌词舞台为核心的沉浸式音乐播放器。

它不是传统意义上的“歌单播放器”，而是把音乐播放、节奏分析、粒子画布、歌词空间和视觉镜头组合成一个更接近现场感的桌面体验。

## 当前版本

当前版本：`0.9.9`

状态：早期公开版本准备中，功能仍在快速迭代。

## 核心特性

- 沉浸式粒子画布与浮空粒子层
- 基于节奏的电影镜头视觉系统
- 面向长播客和 DJ 曲目的 DJ 视觉模式
- 歌词舞台、自定义歌词、歌词位置与视觉控制
- 自定义专辑封面上传与裁剪
- 3D 歌单架与歌单队列
- 网易云音乐账号、搜索、歌单、播客等体验接入
- QQ 音乐搜索、登录态与音源补充接入
- GitHub Releases 更新检测与下载入口

## 使用说明

首次公开发布后，Windows 用户可以在 GitHub Releases 中下载安装包。

当前源码目录是 Electron 应用源码，不建议直接下载 `win-unpacked` 目录作为正式分发包。

## 开发运行

```bash
npm install
node server.js
```

桌面版入口由 Electron 主进程加载本地服务。正式打包流程会在后续版本补充。

## 第三方音乐平台说明

Mineradio 不是网易云音乐、QQ 音乐或腾讯音乐娱乐集团的官方客户端，也不隶属于任何音乐平台。

项目中的第三方平台接入仅用于个人学习、本地客户端体验和用户自有账号的播放辅助。请遵守对应平台的用户协议、版权规则和会员权益规则。项目不会提供绕过付费、绕过会员、破解音质或重新分发音乐内容的能力。

## 用户数据与隐私
